# GEngine (Project IGTTS0420)

**"Simple Atoms, Infinite Possibilities. High-Performance UI for the Next Decade."**

GEngine is a high-performance, native C++ UI engine designed for extreme data density and low-latency visualization. 
It combines the raw power of hardware-accelerated rendering with the flexibility of Script engine.
